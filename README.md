# Iris Flower Classification

This is a beginner Machine Learning project that predicts iris flower species using ML algorithms.

## Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
The project uses the built-in Iris dataset from sklearn.

## Features
- Data visualization
- Model training
- Accuracy calculation
- Confusion matrix
- Multiple ML models

## Accuracy
- Logistic Regression Accuracy: 96%
- Random Forest Accuracy: 100%

## How to Run

1. Install required libraries

```bash
pip install -r requirements.txt
```

2. Run the notebook

Open:
```text
notebook.ipynb
```

and run all cells.
